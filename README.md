# Fog-Assisted Health Monitoring System

## Overview

A Java-based patient health monitoring system that simulates Fog Computing concepts for local health analysis and automated alert generation. The application allows patient registration, health record management, monitoring of vital health parameters, and identification of abnormal health conditions.

## Features

* Patient Registration and Management
* Health Record Storage
* Temperature Monitoring
* Heart Rate Monitoring
* Blood Pressure Monitoring
* Glucose Level Analysis
* Fog Node Based Health Analysis
* Automated Alert Generation
* Console-Based User Interface

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Collections Framework
* Fog Computing Simulation

* Classes and Objects

## Working

1. Register patient details.
2. Store patient health records.
3. Analyze health parameters such as temperature, heart rate, blood pressure, and glucose level.
4. Perform local analysis through simulated fog nodes.
5. Generate alerts when abnormal health conditions are detected.
